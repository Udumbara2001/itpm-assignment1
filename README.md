# ITPM Assignment 1 - Playwright Automation (SwiftTranslator)

This repository contains Playwright (TypeScript) automated tests for:
https://www.swifttranslator.com/

## Requirements
- Node.js (LTS)
- npm

## Install dependencies
Open a terminal in the project folder and run:

```bash
npm install
npx playwright install

```
To run the playwright 

npx playwright test
